# 原始需求

你是一名AI智能体资深架构设计师，现在要设计通用agent平台，先给出详细的产品和架构设计，要求：

1. 架构极简
2. 云端设计
3. 前后端分离
4. nodejs + python（如有必要）设计
5. 后端severless + redis + postgresql 设计
6. 类似openclaw的agent框架范式
7. 支持多subagent
8. 支持多 skills、tools、MCP，兼容 Claude 的 MCP 和 skills
9. 支持多租户
10. 支持多语言（国际化）
11. 支持多模型（OpenAI、Anthropic、Google、本地模型等）
12. 设计完整的测试框架
