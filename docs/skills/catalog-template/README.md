# Anthropic Skills Catalog Template

## 1) 你需要做什么

1. 将本目录文件推送到 GitHub（`GoatGit/semibot`）。  
2. 使用 `raw.githubusercontent.com` 地址作为目录 URL。

## 2) 推荐配置

将 `.env.local` 中的 `ANTHROPIC_SKILLS_CATALOG_URL` 设置为：

`https://raw.githubusercontent.com/GoatGit/semibot/main/docs/skills/catalog-template/skills-catalog.json`

## 3) 说明

- `skills-catalog.json`：安装列表入口文件。  
- `text-editor-manifest.json`：单个 skill 的 manifest 示例。  
- 你可以按同样结构新增更多 skill 条目。
