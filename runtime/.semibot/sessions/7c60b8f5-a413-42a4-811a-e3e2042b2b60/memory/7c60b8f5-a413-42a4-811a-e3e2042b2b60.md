## 重要历史节点：

1. **核算体系变革**：1985年中国开始从苏联的物质平衡表体系(MPS)转向联合国推荐的国民经济核算体系(SNA)

2. **正式实施**：1992年8月国务院正式通知实行SNA体系，GDP成为中国最重要的经济指标

3. **快速增长期**：1993年经济加速增长，境外投资大幅增加，设立了2000多个特别经济区

4. **宏观调控**：1994-1996年实施宏观调控，通货膨胀率从17%降至8%

5. **新世纪发展**：2000年后经济持续快速增长，2020年突破100万亿元大关

## 数据特点：
- 2017年起国家统计局改革GDP核算发布制度，从三步调整为两步
- 数据经过最终核实，如2024年数据经核实后比初步核算减少1018亿元
- 采用不变价格计算实际增长率，消除物价因素影响
